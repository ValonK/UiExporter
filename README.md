# Ui Exporter

Export any UI Element + Data, from any Application.

## Requirements

- Windows 11 or Windows 10 v1903 (18362) or newer
- .NET 6.0 Desktop Runtime [Download](https://dotnet.microsoft.com/en-us/download)
